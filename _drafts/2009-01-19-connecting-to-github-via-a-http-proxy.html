In summary, Git is...

Git is an open source, distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

Every Git clone is a full-fledged repository with complete history and full revision tracking capabilities, not dependent on network access or a central server. Branching and merging are fast and easy to do.


Git uses two different protocols for pulling and pushing. For pulling, it uses its own git:// protocol or http://. The pulls are quite dumb and are not authenticated. This means that once shared, the repositories are accessible for reads. Push operation is generally implemented using th SSH protocol. This protocol ensures that the communication is not only authenticated but also cryptographically secure.
